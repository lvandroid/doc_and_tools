# VSCode插件设置

## 一、设置

### iterm2和zsh设置

    # 设置iTerm为默认的terminal
    "terminal.external.osxExec": "iTerm.app",
    #设置字体
    "terminal.integrated.fontFamily": "Source Code Pro for Powerline",
    #设置shell
    "terminal.integrated.shell.osx": "/bin/zsh",
